# Instructions for agents

- Use `make help` to find available development targets
- Before committing `.py` changes, run `make format` to format
- Don't edit `pyproject.toml` directly, use `uv add` or `uv add -dev` to add dependencies
- Don't edit `package.json` directly
- Before committing `.ts` `html` `.svelte` changes, run `make format` to lint
